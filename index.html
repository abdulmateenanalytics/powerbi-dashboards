Freelance Project

PDF Data Automation & Power BI Query Stabilization

This real-world freelance project focused on fixing recurring Power BI query failures caused by changing PDF report formats. The client was downloading weekly PDF reports from a website and storing them in OneDrive folders for reporting.

I first performed root cause analysis to identify exactly where and why the Power Query steps were breaking. The issue was that different PDF structures were being processed through one transformation logic. I redesigned the solution by creating separate queries for each PDF format group and then appending the cleaned outputs into one final consolidated dataset.

  • Root cause analysis of broken Power Query steps
  • Separate query logic for different PDF structures
  • Folder-based PDF data processing
  • Append queries into one final reporting dataset
  • Reduced manual troubleshooting and improved stability
